# NAME

| cicdaemon -- CIC management daemon

# SYNOPSIS

| cicdaemon
| cicdaemon bootstrap <option>

# DESCRIPTION

**cicdaemon**(1) is used to automatically configure and check the integrity of UNIX-like hosts. This script will only ever allow **root** or the **cicdaemon** account modify it.
